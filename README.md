# 🧠 Basket Goal Puzzle Solver

A Python project that solves a custom grid-based puzzle using multiple search algorithms.

---

## 📌 Overview

This project models a puzzle game as a state-space search problem and applies different algorithms to automatically find a solution.

---

## 🚀 Features

- Grid-based puzzle environment  
- Multiple solving strategies:
  - DFS
  - BFS
  - UCS
  - A*
  - Hill Climbing  
- Manual and automated gameplay  
- Solution path reconstruction  

---

## ⚙️ How It Works

The system:

1. Represents the board as a state  
2. Generates possible moves  
3. Applies search algorithms  
4. Finds a path to the goal (ball → basket)  

---

## 🛠️ Tech Stack

- Python  
- NumPy  
- Deque / PriorityQueue  

---

## ▶️ Run

```bash
python main.py
```
🎯 Goal

Move all balls into baskets using valid moves while navigating obstacles.
